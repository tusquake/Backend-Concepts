How to Get an Access Token

Step 1: Authorization Request

Visit this URL in your browser to start the OAuth flow:

http://localhost:8080/oauth2/authorize?response_type=code&client_id=demo-client&redirect_uri=http://localhost:8080/authorized&scope=openid%20profile%20read%20write

Step 2: Get Token (using cURL)

After authorization, exchange the code for a token:

curl -X POST http://localhost:8080/oauth2/token \ -u demo-client:secret \ -d "grant_type=authorization_code" \ -d "code=YOUR_AUTH_CODE" \ -d "redirect_uri=http://localhost:8080/authorized"

Step 3: Use Token

Use the access token to call protected endpoints:

curl http://localhost:8080/api/user/profile \ -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
